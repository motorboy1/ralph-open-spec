# Plan

## Scope
- README.md의 헌법 불변성 관련 문구 1개만 교체한다.

## Checklist
- README.md에서 불변성 문구 1개만 식별
- 헌법 v1.4와 충돌 없는 대체 문구 작성
- 변경 전/후 diff가 명확히 보이도록 기록

## Procedure
1) README.md에서 대상 문구의 정확한 원문을 확인한다.
2) 헌법 v1.4와 충돌이 없도록 대체 문구를 작성한다.
3) README.md에서 해당 문구만 교체하고, diff를 확보한다.
4) 변경 결과가 다른 문구/구조에 영향이 없는지 점검한다.

## Outputs
- 02_EXECUTE.md: 실제 변경 로그 + diff 포함
- 03_VALIDATE.md: 충돌 해소 여부 확인 결과
